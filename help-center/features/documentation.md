Keep your team's knowledge organized and accessible with eHUB's documentation features.

## Creating Documents

Create structured documentation:
1. Go to the Documents section
2. Click "New Document"
3. Choose a template or start from scratch
4. Use the rich text editor to format content

## Document Organization

Organize documents effectively:
- Create folders and subfolders
- Use tags for categorization
- Set document status (draft, review, published)
- Create document collections

## Markdown Support

Write documentation using Markdown:
- Headers and formatting
- Lists and tables
- Code blocks with syntax highlighting
- Links and references
- Images and media

## Version Control

Track document changes:
- View revision history
- Compare versions
- Restore previous versions
- See who made changes and when

## Sharing and Permissions

Control document access:
- Share with specific team members
- Set read/write permissions
- Generate public share links
- Export to PDF or other formats
