eHUB's project management features help you organize and track work efficiently.

## Creating Projects

To create a new project:
1. Navigate to the Projects section
2. Click "New Project"
3. Enter project name and description
4. Set start and end dates
5. Assign team members

## Project Views

Choose from multiple views:
- **List View**: See all projects in a list
- **Board View**: Kanban-style board for task management
- **Calendar View**: Visualize project timelines
- **Timeline View**: Gantt chart for project planning

## Task Management

Within each project, you can:
- Create and assign tasks
- Set priorities and due dates
- Add labels and categories
- Track progress with subtasks
- Attach files and documents

## Collaboration

Work together effectively:
- Comment on tasks and projects
- @mention team members
- Share updates and status reports
- Track changes and activity history
