Solutions to common problems you might encounter.

## Login Issues

**Problem**: Cannot log in to your account

**Solutions**:
- Verify your email and password are correct
- Check if Caps Lock is enabled
- Try resetting your password
- Clear browser cache and cookies
- Disable browser extensions temporarily

## Performance Issues

**Problem**: eHUB is running slowly

**Solutions**:
- Check your internet connection
- Close unnecessary browser tabs
- Clear browser cache
- Try a different browser
- Disable browser extensions

## Sync Issues

**Problem**: Changes not syncing across devices

**Solutions**:
- Refresh the page
- Check internet connectivity
- Log out and log back in
- Verify sync settings are enabled
- Contact support if issue persists

## Upload Issues

**Problem**: Cannot upload files or images

**Solutions**:
- Check file size limits
- Verify file format is supported
- Try a different file
- Check available storage space
- Ensure stable internet connection

## Contact Support

If you continue to experience issues:
- Email: support@studio27.se
- Include: Description, screenshots, browser/device info
- Expected response time: 24-48 hours
