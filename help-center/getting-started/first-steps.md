After signing into eHUB, follow these steps to set up your account and start using the platform.

## 1. Complete Your Profile

Navigate to your profile settings to add:
- Profile picture
- Contact information
- Notification preferences

## 2. Create Your First Project

1. Click on "Projects" in the main navigation
2. Select "New Project"
3. Fill in the project details
4. Choose team members to collaborate with

## 3. Explore the Dashboard

The dashboard provides an overview of:
- Active projects
- Recent activities
- Upcoming tasks
- Team notifications

## 4. Set Up Integrations

Connect eHUB with your existing tools to streamline workflows. See the Integrations section for more details.
