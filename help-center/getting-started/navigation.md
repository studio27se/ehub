Learn how to navigate eHUB efficiently and find what you need quickly.

## Main Navigation

The main navigation bar provides access to:
- **Dashboard**: Your overview and activity feed
- **Projects**: All your projects and tasks
- **Documents**: Shared documentation and files
- **Settings**: Account and system configuration

## Search Functionality

Use the search bar (Ctrl/Cmd + K) to quickly find:
- Projects
- Documents
- Help articles
- Team members

## Keyboard Shortcuts

Speed up your workflow with these shortcuts:
- `Ctrl/Cmd + K`: Open search
- `Ctrl/Cmd + N`: Create new project
- `Ctrl/Cmd + /`: Open help center
- `Esc`: Close dialogs and modals

## Mobile Navigation

On mobile devices, access the menu via the hamburger icon in the top-left corner.
